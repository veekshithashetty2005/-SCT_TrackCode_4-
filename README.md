This project is a simple To-Do web application developed using HTML, CSS, and JavaScript. It is designed to help users organize their daily tasks in an easy and effective way.

The application allows users to add new tasks to a list and manage them properly. Tasks can be edited, marked as completed, or deleted when they are no longer needed.

Users can also set a date and time for each task, which helps in better planning and task tracking. The interface is clean, simple, and user-friendly, making it easy for anyone to use.
